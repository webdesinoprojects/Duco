## ðŸ“¸ Image Upload Tips

### Using Upload Button
1. Click "ðŸ“¤ Upload" button
2. Select image from computer
3. Image uploads to ImageKit
4. URL auto-fills in field

### Using URL
1. Paste image URL directly
2. Must be valid HTTPS URL
3. Image must be accessible

### Recommended Image Sizes
- **Hero Image:** 1200x600px (16:9 ratio)
- **Side Cards:** 400x400px (square)
- **Middle Banner:** 1200x400px (3:1 ratio)
- **Promo Cards:** 600x400px (3:2 ratio)
